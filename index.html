Welcome to OpenPlanner
OpenPlanner runs AI entirely on your device using Ollama.
Let's get you set up — it only takes a few minutes.
1
Install Ollama
Ollama is a free, open-source local AI runner. Click below to download it, install it, then come back here.
2
Download Llama 3 (~4.7 GB)
Once Ollama is running, OpenPlanner will automatically download the Llama 3 model for you.
3
All Set!
OpenPlanner is ready. Everything runs locally — no subscriptions, no data sharing, no costs.
